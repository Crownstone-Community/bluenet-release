template<class T, int U, class V>
void CyclicStorage< T, U, V >::loadFromStorage ( )
inline

A value can be stored at different locations in FLASH.

The size of the buffer is given by the template parameter U. The loadFromStorage() function iterates through the values and sets the one with the largest sequence number and the sequence number itself in fields of the class.

The load function is automatically called from the constructor. After a store() it is not necessary to call loadFromStorage() before calling read().

If we calculate diff = ([(i-1)U]+U-[i])U it would be: [30-13 29-14 16-15 17-16 18-17 19-18 26-19 27-26 28-27 29-28]U = [1 1 1 1 1 1 7 1 1 1]. The highest value is 3.
