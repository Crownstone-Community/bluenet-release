Detailed Description

MasterBuffer is a byte array with header.

The MasterBuffer is used to put in all kind of data. This data is unorganized. The MasterBuffer can also be accessed through more dedicated structures. This allows to read/write from the buffer directly ScanResults or other types of sophisticated objects.

The disadvantage is that the data will be overwritten by the different accessors. The advantage is that the data fits actually in the device RAM.
