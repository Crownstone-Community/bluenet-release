31 /* Power Switch
32  *
33  * The Switch class governs the two (separate) pathways that enable power to the load (switch ON)
34  * or disable power to the load (switch OFF)
35  * The two pathways are:
36  * * the Relay (for high current loads but slow switching times)
37  * * the IGBTs (for low current loads but fast switching times)
38  * Switching the IGBT pathway is done through Pulse Width Modulation (PWM)
39  *
40  * #define variables used in this class
41  * - PWM_DISABLE If this is defined, the Switch class will ignore all attempts to switch PWM
42  * - CONFIG_PWM_PERIOD Time between consecutive switch ON actions during PWM dim cycle
43  * - CONFIG_RELAY_HIGH_DURATION Duration of time in which power is supplied to the relay in order to switch it
44  *
45  * */
